This is a help document for the Mole Game. It helps you unerstand the game and then play it!

Game Overview:

Given below will be shown a grid, which upon clicking on 'Start' button fills the cells with 'Red' color. If you click on the 'Red' colored cell, it turns into 'Gray' color. If you could turn all cells of the grid into 'Gray' color, within the given time, you win. But if you could not, don't worry. Just try one more time. To play the game again, please click on the 'Reset' button. This will reset the grid. Then click 'Start' to play once again. You'll be given a total of '3' chances to finish the game.

 

Given below is the step wise instructions for playing the game.

Step 1: Select the game level:
  1. Select the 'Game Level' from the dropdown menu followed by clikcing on the 'Select' button.
  2. Following are the grid dimensions: 4x4, 5x5, 6x6.
  3. You can select your custom level as well. For this:
     
    1. Select 'Advance' as the 'Game Level' followed by clicking on the 'Select' button.
    2. There'll be pop-up asking for the grid dimensions. You're supposed to enter the square grid dimensions:
                        like 2x2, 3x3 or 10x0 etc. It should be NxN.
    3. Once you enter the proper grid dimension, the grid will be drawn and the game is ready to be played.
Step 2: Start the game:
  1. Once the grid is drawn, the 'Start' and 'Reset' buttons are shown.
  2. The 'Start' button starts the game and the 'Reset' button is for playing another session.
Step 3: Playing the game:
  1. Once you click on the 'Start' button, the grid starts randomnly filling its cells with the 'Red' color.
  2. Click on the 'Red' colored cell to turn it to 'Gray' color cell.
  3. Once all the cells in the Grid are colored with 'Gray' color, the game finishes.
    1. If you finish the game within given time, you win.
    2. If you could not finish the game, you can try it one more time (a total of 3 such trials are allowed).
    3. To play it for the next time,
      1. Click on the 'Reset' button.
      2. This will reset all cells of the grid.
      3. Once again you can start playing by clicking on the 'Start' button as per the steps mentioned in Step#2 and #3.
  4. If you cannot finish the game in 3 chances, you lose.
